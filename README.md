# CA2021Fall

## Testing for CA hw.
### Structure of dir
```
hw_no/
├── generate.py
├── input
│   └── sample.txt (input files)
├── output
│   └── sample.txt (output files)
├── bin
│   └── your binary file
└── test.py
```
- Prerequisite: an execution file in bin.
- Usage:
```
  cd hw_no
  python3 test.py
```
